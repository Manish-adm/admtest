How Follow-up Status Works

Status is calculated automatically from the follow-up date set when logging a visit.

Done
When: Employee clicked ✏️ Update and set Reminder Status to Done or Sent.
Upcoming
When: Follow-up date is in the future and not marked Done.
Due today
When: Follow-up date is today. Action needed now!
Overdue >14d
Red: More than 14 days overdue. Urgent action needed.
Overdue 8–14d
Orange: 8 to 14 days overdue. Follow up soon.
Overdue 1–7d
Yellow: 1 to 7 days overdue. Getting behind.
No follow-up
When: No follow-up date was set during the visit.
To mark Done: All Visits → click ✏️ Update → set Reminder Status to Done.
