❓ How Follow-up Status Works
Status is calculated automatically from the follow-up date set during the visit entry.
🟢 Done
How it becomes Done: When the employee opens the visit form via the ✏️ Edit button and sets the "Reminder Status" field to Done or Sent. This confirms the follow-up was completed.
🔵 Upcoming
When: A follow-up date is set in the future (after today) and status is not Done. The employee has a scheduled follow-up to do.
🟡 Due Today
When: The follow-up date is exactly today and status is not Done. Action needed today!
🔴 Overdue
When: The follow-up date has passed (before today) and status was never set to Done. The number in brackets (e.g. 5d) shows how many days overdue.
⚪ No Follow-up
When: No follow-up date was set during the visit entry. The visit was informational or no action was needed.
⚡ To mark a follow-up as Done: Go to All Visits → find the entry → click the ✏️ Edit button → update "Reminder Status" to Done in the Google Form.
